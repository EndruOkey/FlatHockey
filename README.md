# FlatHockey V4

Fresh server-authoritative POND-only prototype.

## Dev

```bash
npm install
npm run dev
```

- Client: `http://localhost:5173`
- Server WS: `ws://localhost:8080/ws`

## Build

```bash
npm run build
```

## Start (prod-like server)

```bash
npm run start
```

Server serves `client/dist` if built.
